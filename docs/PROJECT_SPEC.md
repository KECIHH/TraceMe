# Project Spec

TraceMe 是一个个人自用旅行规划网站，目标是帮助使用者在本地整理旅行计划、预算、文件资料和备份记录。项目默认面向单人或家庭内部使用，不以公开多租户服务为目标。

## 技术栈

- Next.js App Router 提供页面和 API 路由。
- TypeScript strict mode 保持类型约束。
- Tailwind CSS 负责界面样式。
- Prisma 访问 SQLite 数据库。
- Vitest 覆盖单元测试。
- Playwright 覆盖端到端测试。
- Docker 文件仅作为后续部署探索的基础。

## 隐私原则

- 默认本地运行，不公开暴露到公网。
- 旅行证件、票据、行程文件等资料应只保存在受信任设备上。
- 测试时使用虚构数据，不使用真实身份证、护照、签证、银行卡或完整票据。
- 后续涉及文件上传时，应默认采用最小权限、可删除、可备份的设计。

## 安全原则

- API Key、会话密钥、加密密钥只放在环境变量中。
- 数据库文件、上传文件和备份文件不进入 Git。
- 管理员初始密码必须在本地环境中修改，不使用默认值长期运行。
- 不把敏感文件、真实证件或私密行程发送给 AI 工具处理。
